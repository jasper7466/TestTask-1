Слайсер

Утилита для удаления повторяющихся букв в словах.

Правила:

  • Можно вводить сколько угодно слов, разделитель - запятая
  • Один элемент - одно слово. Лишние пробелы будут удалены
  • Разрешены буквы и цифры, кроме спец-символов... ¯\_(ツ)_/¯
  • Регистр не важен, можно писать "заборчиком"
